## Project Workflow

### 1. Data Loading
- Imported dataset into Python.
- Performed initial inspection of data structure and quality.

### 2. Exploratory Data Analysis (EDA)
- Checked data types and summary statistics.
- Identified missing values and outliers.
- Analyzed trends, distributions, and relationships.

### 3. Data Cleaning
- Handled missing values.
- Removed duplicates.
- Standardized formats and corrected inconsistencies.

### 4. SQL Analysis
- Loaded cleaned data into the database.
- Wrote SQL queries to:
  - Aggregate metrics
  - Identify trends
  - Perform category-wise analysis
  - Generate business insights

### 5. Dashboard Development
- Created interactive Power BI dashboards.
- Added filters, KPIs, charts, and visual summaries.

### 6. Reporting
- Summarized findings in a professional report.
- Created a presentation using Gamma.
